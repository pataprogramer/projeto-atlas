## O que é o Atlas

O **Projeto Atlas** é minha jornada pessoal para me tornar um engenheiro de IA e software de alto nível — documentada, versionada e construída em público, um capítulo de cada vez.

Não é um curso comum. É uma trilha estruturada como um RPG: capítulos, missões, chefões e XP, onde cada tecnologia só entra em cena quando o projeto realmente precisa dela. Nada é estudado sem propósito.

## A visão final

O objetivo é evoluir este repositório, ao longo dos próximos meses, até se tornar uma **plataforma de automação empresarial baseada em múltiplos agentes de IA**, com:

- 🧠 Orquestrador central de agentes
- 💰 Agente Financeiro
- 📈 Agente Comercial
- 🗂️ Agente Administrativo
- 👨‍💻 Agente Programador
- 🧩 Sistema de memória e conhecimento próprio (RAG)
- 🗄️ Banco de dados relacional e vetorial
- 🌐 API REST e Interface web (painel administrativo)
- ⚙️ Filas, cache e processamento paralelo
- 🎯 Roteamento automático do melhor modelo de IA por tarefa
- 📊 Logs, métricas e observabilidade completa

## Como esta jornada está estruturada

O aprendizado segue um roadmap próprio, o **Mapa Mestre do Projeto Atlas**, dividido em 12 capítulos + um boss final, cobrindo desde fundamentos (terminal, Git, Python) até sistemas multiagentes de produção.

Cada capítulo termina com um projeto-chefão que integra tudo o que foi aprendido — e cada um deles vira, aos poucos, uma peça real da plataforma final.
